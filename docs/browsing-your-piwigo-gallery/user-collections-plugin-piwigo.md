# Create collections (User Collections plugin)

## What is the User Collections plugin?

The User Collections plugin allows users (administrators or not) who are logged into your gallery to create selections of files, to find them again easily, or share them (including with people who don't have an account in your Piwigo gallery).

Each user can create an unlimited number of collections.

To use this feature, you need to activate the **User Collections** plugin.

!!! info "If you are a piwigo.com customer, this plugin is only available from the Enterprise plan and higher"

## Set the permissions associated with collections

Once the User Collections plugin has been activated, you need to go to the plugin's configuration by clicking on "Configuration".

The Configuration tab lets you set the users' rights:

- If they are allowed to make their collections public (meaning, accessible from a share link, including to people who don't have an account in your gallery);
- If they can send their public collections via email;
- If they can send their collections to an administrator.

## How do I create collections?

Once the plugin has been activated, users who are logged in can add photos to a collection:

- From the Album page, by hovering the mouse over a photo's thumbnail
    
    ![Ajouter à collection.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-c281eb3b.jpg)
    
- From the photo page, using the star-shaped icon located in the tool bar
    
    ![Pas de collection.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-c6555429.jpg)
    

You can add a photo to an existing collection, or create one in the process.

All collections created by a user are visible through the Collections menu. For each collection, the number of photos it contains is displayed.

![Collection 1.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-6074b19f.jpg)

The Collections page lists all collections created by the user, lets you open them (to share them for example), or delete them.

![Page collections.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-dc097b1b.jpg)

## How do I share a collection?

When visiting a collection's page, several icons specific to the collection are available in the tool bar (depending on the permissions set in the plugin's configuration).

### Share a link to the collection

![collection.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-383da7f0.png)

The share icon opens a window which allows you to generate a secure share link. If you wish, you can customize the end of the URL or keep the URL generated by default.

If you want, you can also set a password to access this collection, as well as an expiry date.

![Partage collection.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-fe19ab41.jpg)

Once the sharing link has been created, a window lets you claim the generated link to share it.

![Lien collection.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-6ed72494.jpg)

The generated link only lets people view the collection: it doesn't grant access to the private albums' content.

![Collection sunset.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-29866a44.jpg)

### Send a collection via email

![collection-mail.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-ca8f3da8.png)

The "email" icon opens a window which directly lets you share the collection via email.

Depending on the rights set in the plugin's configuration, you can send a selection through email to an outside person, to an administrator, or both.

![Envoi collection email.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-c6f38b4a.jpg)

The email is automatically sent to the recipient.

![Mail collection reçu.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-6fb0a142.jpg)

## Manage a collection in the gallery

![collection - copie 2.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-4f3093be.png)

On a collection's page, the pen-shaped icon allows you to edit its name and add a description.

![Modifier collection.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-30e13078.jpg)

![collection - copie 3.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-67372f9a.png)

The broom-shaped icon lets you empty the content of a collection.

![collection - copie 4.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-34926fec.png)

The trash can-shaped icon lets you delete the collection.

## Manage collections in the administration

Administrators can have access to collections created from the User Collections plugin's configuration.

A search engine lets you filter collections by name and by user.

![Paramètres plugin.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-6208ae8d.jpg)

From this screen, the administrator can find the collections created, edit them, export them in CSV format, and delete them.
