# The Modus theme

## Presentation of the Modus theme

Modus is the default theme activated on Piwigo.

It is a modern theme which offers a guaranteed compatibility with most plugins.

Its characteristics:

- *responsive* theme compatible with all screen sizes
- large square thumbnails for the albums
- horizontal menu bar
- automatic detection of the photo size to display
- compatibility with "high pixel density" screens (Retina type) using a specific photo size selection
- SVG icons (icons that are always clear no matter the level of zoom).

![Page accueil Modus.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-041139c5.jpg)

## Customizing Modus’ colors

Modus offers 18 different color sets.

To choose yours, once you have activated the Modus theme in the administration (Configuration > Themes), click on the Configuration button.

![Activation thème Modus.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-a9bb2538.jpg)

You can then view the different *skins* or color sets available.

Click on the "Preview" button if you want to expand the preview.

Click on the style of your choice, then on "Save Settings": you can go see what this looks like in your gallery!

![Liste thèmes Modus.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-80c0803e.jpg)

## Modus options

The Modus theme configuration offers other options.

![Réglages miniatures.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-bb92d8e2.jpg)

### Album thumbnail customization

By default, Modus displays square thumbnails with 250px sides for albums.

You can deselect the "square thumbnails" option: the albums will then be displayed as large blocks, allowing for the albums’ descriptions to be displayed.

![Option annulée.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-7e524849.jpg)

You can also choose to keep square thumbnails, and edit the size of these thumbnails.

### Photo thumbnail customization

By default, on a list of photos, Modus displays XXS-sized thumbnails.

![Miniatures normales.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-ed7f3dbb.jpg)

If you want, you can edit the default thumbnail size.

Here are some examples.

- Thumbnails in “thumbnail” format:
    
    ![Miniatures miniatures.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-908a6eb1.jpg)

### Adding a banner

You can select the "Display page banner" option on Modus.

This will display a banner at the top of your gallery's menu.

![Bannière.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-9cf5c79c.jpg)

The content of this banner (text or image) can be edited from the Configuration > Options screen in Piwigo's administration.

![Réglages bannière.jpg](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/07/03/20260703190344-c72ee89c.jpg)
